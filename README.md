## 🚀 快速开始

### 1. 直接使用
1. 下载所有文件到本地
2. 配置EmailJS（参考 `EmailJS-Setup.md`）
3. 在浏览器中打开 `index.html`

### 2. 本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .

# 访问 http://localhost:8000
```

### 3. 部署到服务器
1. 将所有文件上传到服务器
2. 配置EmailJS
3. 访问域名即可

## ⚙️ 配置说明

### EmailJS配置（重要！）
1. 注册 [EmailJS](https://www.emailjs.com/) 账号
2. 创建邮件服务
3. 设置邮件模板
4. 获取公钥、服务ID、模板ID
5. 在 `script.js` 中替换配置值
6. 详细步骤请参考 `EmailJS-Setup.md`

### 自定义配置
- **联系方式**: 修改 `index.html` 中的联系信息
- **服务内容**: 更新服务卡片的内容
- **作品集**: 替换作品展示的内容和图片
- **颜色主题**: 修改 `styles.css` 中的CSS变量
