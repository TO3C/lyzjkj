# 流云智炬 · 合同条款与付款

## 合同体系

### 合同模板（2份）
1. 网站开发服务合同 — 含数据保护、服务过渡、分包限制条款
2. 小程序开发服务合同 — 含微信规则变更、审核次数上限条款

### 合同附件（3份）
1. 交付清单 — 项目交付物清单签收确认
2. 设计稿确认单 — 设计稿评审确认
3. 验收确认单 — 项目验收确认

### 法律文件（4份）
1. 服务条款 — 对外服务通用条款
2. 保密协议 — 项目保密协议
3. 售后服务承诺 — 售后服务标准承诺书
4. 隐私政策 — 官网隐私政策

## 付款方式

### 标准付款方案

| 项目金额 | 付款方案 |
|---------|---------|
| ≤ ¥10,000 | 50%首付 + 50%验收后付 |
| ¥10,001 ~ ¥30,000 | 40%设计启动 + 30%开发中期 + 30%上线前 |
| > ¥30,000 | 30%设计启动 + 30%开发中期 + 20%测试完成 + 20%上线前 |

首付款在合同签订后3个工作日内支付。尾款在验收合格后3个工作日内支付。

支持：银行转账 / 支付宝 / 微信支付

### 催款流程

验收通过 → 发送验收确认单+尾款通知 → 3个工作日内支付 → 超期开始计算违约金 → 超期7天提醒 → 超期15天正式催款函

## 知识产权归属

| 项目 | 归属 |
|------|------|
| 前端源文件/源码 | 归客户所有 |
| 域名/服务器权限 | 归客户所有 |
| CMS系统/UI框架 | 归乙方所有（流云智炬） |
| 案例展示权 | 归乙方所有（流云智炬），可匿名展示 |

域名用客户信息注册，服务器用客户账号购买。客户拥有网站/小程序的全部所有权。

## 违约责任

| 责任方 | 处理方式 |
|--------|---------|
| 甲方（客户）逾期付款 | 每日万分之一违约金，逾期超期可终止合同 |
| 乙方（流云智炬）延期交付 | 每日万分之一赔偿，上限合同金额5% |
| 甲方原因导致延期 | 乙方有权收取延期费用 |

## 其他核心条款

- 工期起算：合同签订+首付款到账次日起算
- 验收时效：甲方收到验收通知后，逾期未回复视为验收通过
- 争议解决：协商优先，不成则向乙方所在地（三亚）人民法院诉讼
- 保密期限：合同终止后仍有效
- 不可抗力：双方协商解决或终止合同
- 报价有效期：30天

## 签约方式

- 电子签约：法大大/腾讯电子签（推荐）
- 纸质签约：需双方签字盖章
- 签约前务必确认客户已阅读并理解所有条款
- 知识产权条款单独向客户说明
- 电子签约前通过微信/电话确认关键条款
