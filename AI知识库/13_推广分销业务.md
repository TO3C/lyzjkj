# 流云智炬 · 推广分销业务

> 除自研的网站/小程序开发服务外，公司还通过推广分销模式经营以下业务，面向终端客户直接销售，赚取平台佣金。

## 一、大地保险

- 合作方：中国大地保险
- 合作模式：推广分销（佣金制）
- 主要产品：车险、意外险
- 客户类型：个人客户 + 企业客户
- 售后方：大地保险公司
- 产品线：丰富，具体险种和佣金比例待补充

## 二、携程开放平台

- 合作方：携程（Ctrip）开放平台
- 合作模式：推广分销（佣金制）
- 主要产品：旅游套餐、机票、酒店等
- 客户类型：个人客户 + 企业客户（差旅）
- 售后方：携程平台
- 产品线：丰富，具体产品范围和佣金比例待补充

## 三、运营商电话卡业务

- 合作方：中国移动、中国联通、中国电信、中国广电
- 合作模式：推广分销（佣金制）
- 主要产品：各运营商电话卡套餐
- 客户类型：个人客户为主
- 售后方：对应运营商
- 覆盖：四大运营商全覆盖，具体套餐产品和佣金政策待补充

## 分销业务特点

| 特点 | 说明 |
|------|------|
| 轻资产运营 | 无需自研产品，依托大平台资源 |
| 佣金收入 | 按成交单量获取佣金，无需垫资 |
| 客户来源多样 | 可与网站/小程序客户交叉销售 |
| 售后由平台承担 | 保单/旅游产品/电话卡的售后由合作方负责 |
| 产品线丰富 | 各平台产品种类多 |

## 业务协同联动

五条业务线互相引流：

1. **渔村梦想家用户（租客/房东）**：
   - 房东需要宣传 → 推荐网站/小程序开发
   - 租客需要保险 → 推荐大地保险
   - 新来三亚的人 → 推荐电话卡套餐
   - 旅游人群 → 推荐携程旅游套餐

2. **技术开发客户（企业）**：
   - 企业员工福利 → 推荐团意险
   - 企业差旅 → 推荐携程企业差旅
   - 企业通信 → 推荐企业电话卡套餐

## 执行方法

- 渔村梦想家小程序底部导航加"更多服务"，展示保险/电话卡/技术开发
- 公众号底部菜单分栏：租房服务/技术开发/保险通讯
- 所有业务通过同一个微信号承接，朋友圈内容轮播各业务
- 微信给客户打标签（租房/建站/保险），精准推送对应内容
