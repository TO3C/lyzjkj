# 流云智炬 · 服务流程与工期

## 标准合作流程（7步）

1. **需求沟通**（免费）：了解客户的行业、功能需求、预算、参考风格——30分钟内给出初步评估
2. **方案报价**：填写需求调研表，深入沟通——1个工作日内提供正式方案和报价
3. **签约启动**：签订合同+支付首付款——提供项目所需素材和资料
4. **UI设计**：风格调研→首页设计→客户反馈→定稿→内页设计——风格确定1-2天+设计2-3天
5. **开发测试**：前端开发→后端开发→联调→内部测试→功能/兼容/性能/SEO全检
6. **验收上线**：客户验收（3个工作日内）→部署上线→操作培训（后台管理+内容更新+常见问题）
7. **售后服务**：免费售后3个月→7天/30天/90天/180天回访

### 注意事项

- 工期自合同签订+首付款到账次日起算
- 工期不含客户反馈时间（如设计稿审阅、内容确认）
- 小程序另加微信审核7-14天，不计入工期
- 微信审核首次失败免费修改

## 设计流程细节

```
需求确认 → 风格调研（3-5个参考+色彩方案，0.5-1天）
→ 首页设计（Figma/Sketch高保真原型，移动端+PC端双版，1-2天）
→ 客户反馈1（修改意见汇总，1-2天）
→ 首页定稿（修改后终稿，0.5-1天）
→ 内页设计（统一风格，1-2天）
→ 设计确认（客户签署《设计稿确认单》）
```
