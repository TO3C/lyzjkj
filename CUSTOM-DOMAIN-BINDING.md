# 自定义域名绑定指南

## 📋 前提条件

1. **已部署静态网站**：网站已成功部署到 CloudBase（参考 `CLOUDBASE-DEPLOYMENT.md`）
2. **拥有自定义域名**：`www.lyzjkj.top`
3. **域名管理权限**：拥有该域名的 DNS 管理权限

## 🚀 绑定步骤

### 1. 登录腾讯云控制台

访问 [腾讯云控制台](https://console.cloud.tencent.com/)，使用您的腾讯云账号登录。

### 2. 进入云开发环境

1. 在控制台顶部搜索栏搜索「云开发」
2. 点击进入「云开发」服务
3. 选择已部署网站的环境：`lyzjkj-top-7g2o2uud420b1470`

### 3. 配置静态网站自定义域名

1. 在左侧导航栏找到并点击「静态网站托管」
2. 切换到「自定义域名」选项卡
3. 点击「绑定自定义域名」按钮

### 4. 输入域名信息

1. **域名**：输入 `www.lyzjkj.top`
2. **证书配置**：
   - 选择「免费申请腾讯云证书」（推荐）
   - 或选择「已有证书」（如果您已有 SSL 证书）
3. 点击「下一步」

### 5. 验证域名所有权

根据提示完成域名所有权验证，通常有两种方式：

1. **DNS 验证**：在域名服务商处添加指定的 TXT 记录
2. **文件验证**：在网站根目录上传指定的验证文件

### 6. 配置 DNS 解析

在您的域名服务商（如腾讯云 DNS、阿里云 DNS 等）控制台，添加以下解析记录：

| 记录类型 | 主机记录 | 记录值 | TTL |
|---------|---------|-------|-----|
| CNAME   | www     | lyzjkj-top-7g2o2uud420b1470-1302371739.tcloudbaseapp.com | 10 分钟 |

### 7. 等待配置生效

- 证书申请和部署：通常需要 5-10 分钟
- DNS 解析生效：通常需要 10-30 分钟（取决于 TTL 设置）
- 自定义域名绑定：通常需要 10-20 分钟

## 🔧 验证绑定结果

### 方法 1：直接访问域名

在浏览器中访问 `https://www.lyzjkj.top`，如果能正常显示网站，则绑定成功。

### 方法 2：使用 curl 验证

```bash
curl -I https://www.lyzjkj.top
```

如果返回 `HTTP/2 200` 状态码，则绑定成功。

## ⚠️ 常见问题

### 1. 证书申请失败

- **原因**：域名所有权验证失败
- **解决方案**：
  - 检查 DNS TXT 记录是否正确配置
  - 确保域名没有被其他服务占用
  - 尝试重新申请证书

### 2. DNS 解析不生效

- **原因**：DNS 缓存或配置错误
- **解决方案**：
  - 检查解析记录是否正确
  - 等待 DNS 缓存过期（通常 1-2 小时）
  - 使用 `nslookup www.lyzjkj.top` 验证解析结果

### 3. 访问域名显示 404

- **原因**：静态网站默认文档设置错误
- **解决方案**：
  - 确保 `index.html` 已正确上传
  - 在控制台检查静态网站的默认文档设置

### 4. 访问域名显示不安全

- **原因**：SSL 证书未生效或配置错误
- **解决方案**：
  - 等待证书部署完成
  - 检查证书状态
  - 确保域名与证书匹配
