### 2. EmailJS配置（必须完成）

#### 🔑 获取必要信息
1. **注册账号**: [EmailJS官网](https://www.emailjs.com/)
2. **创建服务**: 连接邮箱服务
3. **设置模板**: 创建邮件模板
4. **获取配置**: 
   - Public Key（公钥）
   - Service ID（服务ID）
   - Template ID（模板ID）

#### ⚙️ 修改配置文件
在 `script.js` 中找到并替换：
```javascript
// 第186行左右
emailjs.init("YOUR_PUBLIC_KEY"); 

// 第208行左右  
emailjs.send('YOUR_SERVICE_ID', 'YOUR_TEMPLATE_ID', formData)
```

## 🔧 故障排除

### 常见问题及解决方案

#### 1. 样式显示异常
**问题**: CSS未加载或显示错位
**解决**: 检查CSS文件路径，清除浏览器缓存

#### 2. JavaScript错误
**问题**: 控制台报错，功能异常
**解决**: 检查语法错误，查看浏览器开发者工具

#### 3. 图片显示异常
**问题**: 二维码或其他图片不显示
**解决**: 检查图片路径，确认文件存在

#### 4. EmailJS发送失败
**问题**: 表单提交后无邮件
**解决**: 
1. 检查公钥、服务ID、模板ID
2. 确认EmailJS服务状态
3. 检查邮箱设置

#### 5. 响应式布局问题
**问题**: 移动端显示异常
**解决**: 检查媒体查询，调整断点
